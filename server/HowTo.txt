1. Устанавливаем Node.js (https://nodejs.org/en/)
2. Устанавливаем MongoDB (https://www.mongodb.com/)
3. Запуск сервера игры node server.js (в отдельной консоли, в папке server)
4. Запуск сервера базы данных mongod --dbpath=data (в отдельной консоли, в папке server, не в той где запускается сервер)
5. Запуск игры. Браузер localhost:8080


Возможные проблемы:
- для доступа во внешних сетях может понадобится пробросить порт 8080 на винде и на роутере
